Intro to Combinatorics

Imagine being at a Christmas party. You have 3 friends and 3 gifts, A, B and C to give them. How many ways are there to give these gifts to your friends? Give it a try

Let us draw a table to see all the possible ways to give the gifts to your friends, and say their names are Alice, Bob and Charlie

Alice Bob Charlie
A B C
A C B
B A C
B C A
C A B
C B A

As you can see, there are 6 combinations. Simple right? Not so quick. What if we increased the number of people to 4, or even 5? How long would it take to list all the possibilities?

Clearly, we need another method

The better method

Use this time to think of another method, use your brain to see if you can think of anything. If not, no worries, we will show you!

Let's think about this in another way. Think in Alice's perspective, which gift do you think Alice would like most? Well there are 3 possibilities. Now, after that, which gift do you think Bob would like most? Well, since we have already chosen a gift for Alice, there are now 2 possibilities for Bob. Finally, there is only one possibility for Charlie.
So in conclusion, 3 choices for Alice, 2 for Bob and 1 for Charlie. So, the number of possibilities = 1 x 2 x 3 = 6, which is the answer!

Using this method, imagine the same scenario but now you have 4 friends and 4 gifts. The same question, how many possibilities are there?
