Let me introduce to you a new function! No, literally. You don't get it? The new function is '!'.

So what does this ! mean? Let's see if you can figure it out yourself
1!=1
2!=2
3!=6
4!=24
5!=120

If you couldn't see it, it's fine. However, if you did see it you really are a genius!

n!=n(n-1)(n-2)....1

So, for example 5!=5x4x3x2x1=120
