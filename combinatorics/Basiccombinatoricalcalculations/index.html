The Subfactorial

The Subfactorial is a function which is the factorial, but with added conditions. First, let's define the Subfactorial

!n = Subfactorial of n

Okay, so what is the Subfactorial? Imagine the case where we have n friends and n gifts. But now each of the friends already has a predetermined gift

The Subfactorial tells you how many possible cases there are where none of the gifts are given to the correct person

However, the function for finding the Subfactorial is quite complicated. So, we normally just memorise the values since no competition will ask for large Subfactorials

!1=0
!2=1
!3=2
!4=9
!5=44

Of course, if you really have to find the subfactorials of larger values, remember this pattern:
!(n + 1)=(n - 1)(!n + !(n-1))

The Choose Function

So we have covered permutation, which is finding the number of possibilities of something. Now we will be exploring combinations, or known as binomial coefficients or “choosing”.

Combinations literally means choosing, for example there are 10 points on a circle, and we want to find out how many straight lines can be drawn within the circle with no lines overlapping. This can be written as 10C2, or “choose 2 from 10”.

The formula for nCr is:

n!/r!(n-r)!

Fun fact: nCr is the same as nC(n-r). Remember this formula well as it would help you in a lot of future questions.
