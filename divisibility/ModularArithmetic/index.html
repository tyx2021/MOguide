Modular Arithmetic

Introduction: What is Modular Arithmetic?

In mathematics, modular arithmetic is a system of arithmetic for integers, where numbers "wrap around" when reaching a certain value, called the modulus. The modern approach to modular arithmetic was developed by Carl Friedrich Gauss in his book Disquisitiones Arithmeticae, published in 1801.

This is from wikipedia

So, how does Modular Arithmetic work?

Essentially, Modular Arithmetic is "remainder" but better. If you don't know what a remainder is, a remainder is the amount of stuff left after you divide something. For example 7 ÷ 4 = 1 remainder 3 because 1 x 4 + 3 = 7

Simple enough, now if we use the previous example, we could rewrite it as 7=3(mod 4) because 3 was the remainder when 7 was divided by 4. In general, if x ÷ y = a Remainder(From here on referred as R) z where a is a whole no., y = z(mod x)

Now try this question:What is 10(mod 3)?

10 ÷ 3 = 3 R 1

Therefore 10 = 1(mod 3)

Properties of Modular Arithmetic

One very important property is this:

If x=y(mod z), x = az + y for some integer a. Let this be Identity 1.1

Using this, let's prove our second identity. Sit tight.

Identity 1.2

If x=y(mod z), x-y=0(mod z)

Let a be an integer

Firstly, using identity 1.1, x = az + y. Then x - y = ax + y - y = ax = 0(mod x)

Notice how we let a be an integer, not a positive integer but an integer.

This means that the identity also applies for negative values of mod. But... that makes no sense!

Identity 1.3

-x=y-x(mod y)

This is a follow-up for identity 1.2

So what is a negative mod?!

If we understand this concept, everything from now on will be much easier.

Let's try some things out. Hmmm well 2 = 2(mod 3) obviously. Then 2^2 = 1(mod 3)

So what numbers when squared become 1? Well, 1 squared obviously works but 1 is not equal to 2. That must mean that the other solution, or -1 is 2 in this case. This is the most crucial step

Identity 1.4

x * y(mod z) = x(mod z) * y(mod z)

Let a b, c and d be integers such that x = az + b, and y = cz + d

x * y = (az + b)(cz + d) = acz2 + (ad + bc)z + bd = bd(mod z)
