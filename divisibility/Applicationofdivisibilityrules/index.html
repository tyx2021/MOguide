Excersises for divisibility rules

Q1. Prerequisites: Divisibility rules

The number 20022002 . . . 20022002 is formed by writing 2002 blocks of '2002'.
Find the remainder when the number is divided by 9.
(SMOPS 2002 Round 1 Qn 21)

In every '2002' the sum of digits is 2 + 0 + 0 + 2 = 4. So the sum of 2002 '2002's is 2002 x 4 =8008
The sum of digits of 8008 is 16
The sum of digits of 16 is 7
So the number has a remainder of 9

Q2. Prerequisites: Divisibility Rules

Only one of the following four numbers is a perfect square. Which one is it?
(1) 76186 (2) 750235 (3) 921438 (4) 2660161
(SMOPS 2014 Q.5)

We can see that 76186 is divisible by 2 but not 4, so that must not be the square number. The same applies for 921438. We can also see that 75025 is divisible by 5 but not 25. Hence the answer is (4) 2660161

Q3. Prerequisites: Divisibility Rules

. A 5-digit number written in the form 24abc has the last three digits unknown. If this number is divisible by 3, 4 and 5 respectively, find the greatest possible value that abc can take. (SMOPS 2013 Q.11)

Since 24000 is divisible by 2, 3 and 5, xyz must be divisible by 2, 3 and 5. The largest value of xyz is 990.
